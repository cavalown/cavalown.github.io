    fetch vs. axios

    tags: JavaScript

    fetch API

    • CDN參考資訊

    • Promise 回傳：使用then進一步處理，使用catch處理錯誤。

    • 回傳物件：ReadableStream，是Response.body()的實體。

    • 回傳資料類型處理方法各異：

      • Response.body().arrayBuffer()
      • Response.body().blob()
      • Response.body().formData()
      • Response.body().json()
      • Response.body().text()

    • 要自行判定回傳代碼。

    • 在404時，不會顯示error，所已要真對response.ok 是否為true做檢查。（參考CDN）
